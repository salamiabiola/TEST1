POUR APPRENDRE A FAIRE UN REPOSITORY.
JE VIENT DE MODIFIER POUR VOIR COMMENT APPLIQUER LES MODIF.
1- FAIRE TOUJOURS UN git status POUR VOIR LE STATUT DU DOSSIER.
2- ENSUITE, FAIRE UN git add . (POUR METTRE TOUS LES FICHIERS EN LISTE D'ENVOIE) OU FAIRE UN git add NOM FICHIER
4- APRES CA, IL FAUT FAIRE UN git commit -m "Commentaire" (Pour appliquer les changements)
5- ENFIN, UN git push (Pour envoyer les modifs dans le repository)
